- ls [option] .. files là command để liệt kê thông tin của các files và thư mục  trong thư mục hiện tại
- cd dùng để thay đổi thư mục làm việc hiện tại của terminal
- pwd [-LP] dùng để in ra path hiện tại mà terminal đang ở
- mkdir [option].. dirs dùng để tạo nhiều thư mục 1 lúc, nếu chúng chưa tồn tại
- rm [option] .. files  dùng để xóa / gỡ liên kết các files khỏi filesystem, với thư mục thì cần có option '-r'
- cp [option] .. sources .. dest là tạo bản sao files hoặc thư mục từ nguồn sang đích
- mv [option] .. sources .. dest là di chuyển files và thư mục từ nguồn sang đích
- touch [option] .. files .. cập nhật thời gian truy cập và chỉnh sửa của file, nếu file chưa tồn tại thì tạo file rỗng
- cat [option] .. files là nối nội dung các files hoặc nhận stdin rồi in ra standard output(terminal, pipe hoặc file khác)
- less [option] .. files  dùng cho xem nội dung theo từng trang, có thể cuộn lên/xuống và tìm kiếm trong files
- head [option] .. files in (mặc định) 10 dòng đầu tiên của mỗi file vào standard ouput, nếu có nhiều hơn 1 file thì in thêm tiêu đề tên file
- tail [option] .. files là ngược lại của head; có thể dùng `-f` để theo dõi file log liên tục
- grep [option] dùng để tìm và in các dòng chứa pattern trong một hoặc nhiều file ra stdout
- find [-H] [-L] [-P] [-Olevel] [-D debugopts] [path...] [expression] tìm file/thư mục trong filesystem theo điều kiện và mặc định in đường dẫn kết quả ra stdout
- xargs [option].. command [initial-args] .. dùng để chạy lệnh với các tham số ban đầu và nhiều tham số hơn đọc từ input hoặc 
- awk [POSIX or GNU style options] [--] 'program' file ... là xử lý text theo dòng/cột bằng chương trình awk viết trực tiếp hoặc lưu trong file với option `-f`
- sed [OPTION]... {script-only-if-no-other-script} [input-file]... là stream editor dùng để chỉnh sửa text tự động theo rule trong 1 file và in ra stdout
- sort [OPTION]... [FILE] dùng để in kết quả nội dung đã nối của các file(s) vào standard ouput 
- uniq [OPTION]... [INPUT [OUTPUT]] Lọc các dòng giống nhau nằm cạnh nhau từ iput hoặc stdin, rồi ghi kết quả ra output hoặc stdout
- wc [OPTION]... [FILE]... là đếm bytes, lines, chars.. trong file hoặc stdin
- tee [OPTION]... [FILE]... copy stdin vào mỗi file và in ra stdout
- ps [options] dùng để xem danh sách process đang chạy
- top [options] dùng để xem process đang chạy theo thời gian thực
- htop [options] cũng dùng để xem process đang chạy theo thời gian thực, giống top, nhưng giao diện dễ nhìn và dễ thao tác hơn
- kill [option] PID gửi signal tới process theo PID, thường dùng để yêu cầu tắt, ép dừng, tạm dừng hoặc tiếp tục process
- nice [OPTION] [COMMAND [ARG]...] chạy một command với mức ưu tiên CPU khác; nice càng cao thì process càng ít được ưu tiên CPU
- df [OPTION]... [FILE]... đùng dể xem dung lượng filesystem/partition đang còn trống, đã dùng bao nhiêu 
- du [OPTION]... [FILE]... dùng để xem dung lượng của các files và folders(liệt kê dung lượng đã chiếm của các files và folders con bên trong)
- free [options] dùng để xem tình trạng RAM và swap của hệ thống
- uptime [options] dùng để xem máy đã chạy được bao lâu, hiện tại có bao nhiêu user đang login, và tải hệ thống trung bình
- uname [OPTION]... dùng để xem thông tin hệ thống/kernel
- who [OPTION]... [ FILE | ARG1 ARG2 ] dùng để xem những user/session nào đang đăng nhập vào hệ thống
- chmod [OPTION]... MODE[,MODE]... FILE... dùng để thay đổi quyền truy cập của file/thư mục
- chown [OPTION]... [OWNER][:[GROUP]] FILE... đổi owner và/hoặc nhóm của mỗi file
- umask [-p] [-S] [mode] Đặt mask quyền mặc định được áp dụng ngay khi tạo file/thư mục mới, để tự động bỏ bớt quyền không mong muốn như quyền ghi của group/others 
- tar [OPTION...] [FILE]...dùng để gom nhiều file/thư mục thành một file archive hoặc giải nén/mở archive
- gzip [OPTION]... [FILE]... dùng để nén file bằng thuật toán gzip
- zip [-options] [-b path] [-t mmddyyyy] [-n suffixes] [zipfile list] [-xi list] dùng để gom nhiều file/thư mục và nén chúng thành một file .zip
- unzip [-Z] [-opts[modifiers]] file[.zip] [list] [-x xlist] [-d exdir] dùng để giải nén/mở file .zip ra lại file/thư mục bình thường
- ssh user@ip dùng để đăng nhập và điều khiển một máy khác qua mạng một cách bảo mật
- scp file user@ip path là lệnh để copy file/thư mục qua mạng bằng SSH từ máy bạn đến máy còn lại hoặc ngược lại
- rsync [OPTION]... SRC [SRC]... [USER@]HOST:DEST dùng sync file hoặc folder thay vì copy y nguyên giữa local và local, remote, làm giảm thời gian khi deploy lặp lại,..
- ln [OPTION]... [-T] TARGET [LINKNAME] <DIR> dùng để tạo link/tên khác trỏ tới file hoặc folder đang đứng ở hiện tại
- ln -s dùng tạo link trỏ tới đường dẫn tới file hoặc folder đang đứng ở hiện tại nhưng giống shortcut hơn, nếu file hoặc folder gốc bị rm thì link bị gãy luôn không truy cập vào data 
- env [OPTION]... [-] [NAME=VALUE]... [COMMAND [ARG]...] dùng để xem hoặc chạy command với environment variables 
- export [-fn] [name[=value] ...]  đánh dấu biến shell thành environment variable để các process con sau này có thể kế thừa; không thay đổi environment của process cha
- source [-p path] filename [arguments] sẽ chạy các lệnh từ 1 file trong shell hiện tại
- curl [options...] <url> là lệnh dùng để gửi request tới một URL và nhận response trả về
- wget [OPTION]... [URL]... dùng để tải nội dung/file từ URL về máy qua terminal
- which [options] [--] COMMAND [...] dùng để tìm đường dẫn thật của command mà shell sẽ chạy
- whereis [options] ... [-BMS <dir>... -f] <name> dùng để tìm vị trí của command/program và các file liên quan của nó
- type [-afptP] name [name ...] cho biết shell sẽ hiểu một tên command là alias, function, builtin, keyword hay file executable.

- history [option] dùng để xem lại các lệnh bạn đã gõ trong shell
- alias [-p] [name[=value] ... ] dùng để xem danh sách alias hiện tại có trong shell hiện tại, có thì sẽ dùng để đặt alias cho 1 command trong shell hiện tại
- echo [-neE] [arg ...] dùng để in text/giá trị ra stdout
- printf [-v var] format [arguments] dùng để in dữ liệu theo format giống C, không tự thêm newline; hỗ trợ placeholder như %s, %d, %.2f và có thể lưu kết quả vào biến bằng -v
